# Tetris
Тетрис на masm

Управление:
стоп - пробел
пауза - p
увеличить/уменьшить скорость +/- соответственно
либо на цифры 1 - 9(не numpad)
новая игра - n
повернуть - r
управление - стрелки
уронить - u
выход/назад - esc

фишки:
счет очков (запустить RUSdriver чтоб текст был читаем)
показ следующей фигуры
редактор фигурок - f11
